
	1.	JSTL

			 * taglibs-standard-spec-1.2.5.jar

			(1) core library		: 핵심 기능 제공(변수, 제어문)
			(2) xml library
			(3) fmt library			: 형식 (날짜/시간, 숫자 등)
			(4) sql library
